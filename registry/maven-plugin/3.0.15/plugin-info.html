Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
apicurio-registry:download Download artifacts.
apicurio-registry:merge No description.
apicurio-registry:register Register artifacts against registry.
apicurio-registry:test-update
Deprecated. In favor of using the "dryRun" option of the "register" mojo.

Test artifact against current artifact rules, if an update is possible / valid.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.9
JDK 17

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>io.apicurio</groupId>
          <artifactId>apicurio-registry-maven-plugin</artifactId>
          <version>3.0.15</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>io.apicurio</groupId>
        <artifactId>apicurio-registry-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
