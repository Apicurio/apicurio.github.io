Migration from version 0.2.x to version 1.0.0

Apicurio Studio 1.0.0 includes new features with breaking changes from the previous Apicurio Studio 0.2.x release. This section describes the major changes between Apicurio Studio 0.2.x and version 1.0.0.

Because of the breaking changes in 1.x, there is no automatic upgrade and a migration process is required. This involves moving the data in your existing Apicurio Studio to a new Apicurio Studio instance.
