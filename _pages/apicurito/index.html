What is Apicurito?

We had a need to provide basic API design capabilities in an ultra-slim package, without all of the extra features found in Apicurio Studio - and thus Apicurito was born!

Apicurito is essentially the API editor in Apicurio Studio repackaged as a simple standalone web application. It has no back-end component at all, which makes it incredibly easy to package, deploy, and use. However, the trade-off is that it supports only the core visual editing experience in Apicurio Studio, without the collaboration, persistence, and integration with external systems.

Use Apicurito if all you want to do is load an OpenAPI or AsyncAPI file from your local file system and visually edit it, and then save it back to your disk!
