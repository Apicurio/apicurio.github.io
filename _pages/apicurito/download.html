Getting Started (Docker)

The easiest way to get started with Apicurito is by using Docker to quickly start up an instance, then point your browser to it.

Pull the image

  1. Pull the apicurito-ui docker image from docker hub. 
docker pull apicurio/apicurito-ui:{{ version }}

Run the image

  1. Run the apicurito-ui docker image
docker run -it -p 8080:8080 apicurio/apicurito-ui:{{ version }}

Open Apicurito in your browser

  1. Open your browser to http://localhost:8080
