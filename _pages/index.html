Apicurio Registry

High performance, runtime registry for API designs and schemas

Apicurio Registry is a runtime server system that stores, manages, and governs API definitions and data schemas across their entire lifecycle. It supports OpenAPI, AsyncAPI, GraphQL, Avro, Protobuf, JSON Schema, and more.
